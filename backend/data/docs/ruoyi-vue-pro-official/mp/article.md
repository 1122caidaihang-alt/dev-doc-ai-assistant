# 公众号图文
  
本章节，讲解公众号图文的相关内容，包括两部分：
 
① 在 [公众号管理 -> 图文草稿箱] 菜单中，创建一个图文草稿。如下图所示：
 

 
② 点击【发布】按钮，将图文草稿发布到公众号，成为一个图文记录，展示在 [公众号管理 -> 图文发表记录] 菜单中。如下图所示：
